Projeto PitStop

Desenvolvedor FrontEnd

O Projeto consistia em um site "gameficado" para uma dinâmica de uma instituição educacional, Senac, onde a proposta era permitir que os alunos tivessem acesso a toda programação e escolher alguns das apresentações dos projetos interadores para assistir. Contendo não somente a programação como chat de feedback para cada apresentação, pontuações e podium para quem comparecesse em mais projetos e deixando sua contibuição no chat.

Projeto Orbit

Desenvolvedor FrontEnd e BackEnd

Projeto integrador de finalização do curso, foi desenvolvido um tipo de rede social que abrangeria noticias, campeonatos e troca de produtos, onde a ideia principal era uma rede social onde se podia trocar produtos que não se utilizasse mais, sem fins monetários. Os usuários poderiam trocar seu produto com o outro. O site continha perfil para cada usuário, chat para troca e negociação, uma pagina de noticias, campeonatos e de produtos/itens.
